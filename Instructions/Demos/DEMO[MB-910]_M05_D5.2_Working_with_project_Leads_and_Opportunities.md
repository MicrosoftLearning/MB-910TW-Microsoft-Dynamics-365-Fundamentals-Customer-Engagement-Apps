# 示範： 使用專案潛在客戶和商機

## 指示

1. 如有必要，請開啟 Dynamics 365 Project Operations 應用程式。 

2. 建立專案潛在客戶

	- 逐步解說程序並重點說明專案特定的詳細資料

	- 使用時間表新增活動並管理活動。 

	- 授與專案潛在客戶資格

3. 管理專案商機。

	- 逐步解說新增專案商機明細項目的程序

	- 為商機建立專案報價

	- 關閉專案商機

4. 管理專案報價

	- 修改報價明細項目

	- 關閉專案報價

5. 管理專案合約

	- 檢查專案合約項目

	- 關閉專案報價
